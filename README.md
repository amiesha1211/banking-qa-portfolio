# Banking Application QA Portfolio

This repository showcases my manual QA testing skills using a sample online banking application. It includes a test plan, functional test cases, and bug reports covering common banking features such as user login, account balance, fund transfers, bill payments, and transaction history.

## Skills Demonstrated

* Manual Testing
* Functional Testing
* Regression Testing
* Boundary Value Analysis
* Positive & Negative Test Scenarios
* Bug Reporting
* Test Documentation

**Modules Tested**

* User Authentication
* Account Dashboard
* Balance Inquiry
* Fund Transfer
* Bill Payment
* Transaction History

This portfolio was created to strengthen my QA skills and demonstrate my ability to design test cases and document software defects.
